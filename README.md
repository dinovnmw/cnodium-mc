
## 🌟 What's New in CNODIUM

CNODIUM is a powerful client built on the solid foundation of FDP Client, significantly enhanced with richer features and a highly optimized, modern UI.

* **Smooth UI Animations:** Experience fluid transitions and seamless opening/closing visual effects.
* **Draggable ClickGUI:** Fully customizable workspace—freely drag, drop, and rearrange windows to fit your setup.
* **Feature-Packed:** Loaded with advanced modules and performance tweaks for the ultimate gameplay experience.

---

## 🚀 How to Run the Client

1. **Launch Your Game:** Preparation.
Open your preferred Minecraft launcher, select **Version 1.8.9 with Forge**, and hit **Play**. The newly compiled JAR is already in your `mods` folder, so the CNODIUM engine will load automatically.


2. **Open ClickGUI:** In-Game.
Join any world or server. Press the **Right Shift** key to toggle the main menu.


3. **Custom Keybinds:** Optional.
If you want to change the menu key, open the in-game chat and type: `.bind ClickGui <your_key>`


4. **Drag & Customize:** Explore.
Enjoy ultra-fast module searching, toggle features instantly, tweak advanced settings, and drag the UI elements anywhere on your screen!
