Hướng Dẫn Cài Đặt

Thêm file JAR vào thư mục mods hiện tại để kích hoạt client engine trong game.

1

Tải File Mod JAR

Tải về bản dựng client mới nhất đã được biên dịch sẵn tại kho mã nguồn Github.

Download Mod (.jar) Source Code
2

Di Chuyển File Vào Thư Mục Mods

Khởi chạy Minecraft phiên bản 1.8.9 với Forge. Đưa file `.jar` vừa tải vào đúng thư mục mod của launcher bạn:

%appdata%\.minecraft\mods\

Vào Game & Phím Kích Hoạt
